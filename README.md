# M05 Programming Assignment 1

## Description

This program implements a Binary Search Tree in C++. It includes a function called singleParent() that counts the number of nodes in the tree that have exactly one child.

The program builds binary search trees using predefined test cases and uses recursion to traverse the tree and identify nodes with only one child.


## Inputs

* Integer values inserted into a binary search tree
* Values are provided through predefined test cases


## Outputs

* The number of nodes in the tree that have exactly one child
* Output is displayed in the console


## How to Compile

g++ main.cpp binaryTreeType.cpp -o program


## How to Run

./program


## Test Cases

* 50, 30, 70, 20, 40, 60, 80 - Output: 0
* 50, 30, 70, 20, 60 - Output: 2
* 10, 20, 30, 40 - Output: 3
