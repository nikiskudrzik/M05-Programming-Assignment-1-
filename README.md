# M05-Programming-Assignment-1-

## Description

This program implements a binary search tree and includes a function to count the number of nodes that have exactly one child.

## Inputs

* Integer values inserted into a binary search tree

## Outputs

* Number of nodes with exactly one child

## How to Compile

g++ main.cpp binaryTreeType.cpp -o program

## How to Run

./program

## Test Cases

1. 50, 30, 70, 20, 40, 60, 80 - Output: 0
2. 50, 30, 70, 20, 60 - Output: 2
3. 10, 20, 30, 40 - Output: 3
